# snappy_shell

A minimal shell written in Rust—read commands, run them, repeat.

## Get started

**Prerequisites:** [Rust](https://rustup.rs/) (e.g. `curl --proto '=https' --tlsv1.2 -sSf https://sh.rustup.rs | sh`).

```bash
cargo run
```

Build only: `cargo build`. Release build: `cargo build --release`.
